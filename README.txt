Este projeto compreende a criação de um código de criptografia de arquivo
e um outro código de descriptografia. O objetivo é compreender e exemplificar
um malware (software malicioso) do tipo ransomware. Criptografa-se o arquivo 
de texto "teste.txt" e, em seguida, descriptografa-se o mesmo arquivo, 
reestabelecendo-se, portanto, o arquivo original com seu conteúdo legível.

Este projeto abarca:

- Sistema operacional Linux (distro Kali)

- Linguagem python
